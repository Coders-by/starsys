## 跑起来

```bash
npm install
echo "GEMINI_API_KEY=你的key" > .env.local   # 没有 key 也能跑，小九会落到离线兜底
npm run dev
```

打开 `http://localhost:3000` —— 默认是手机壳尺寸（千岑还在纠结要不要做桌面适配）。

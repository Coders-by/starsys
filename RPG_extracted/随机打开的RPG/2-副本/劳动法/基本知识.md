# 三、离职证明、离职审批流程、”离职申请表“

1. 公司开具离职证明是法律要求，不得已任何理由拖延或拒绝。有时候公司说要交接完工作再给离职证明，是不合法的。
2. 在任何情况下（即便是自己主动离职），都不需要签署任何所谓的”离职审批流程“？

3. 不存在”离职申请“，有的只是”离职通知“。劳动法赋予劳动者无责辞职的主动权。


# 四、劳动合同解除通知书、解除劳动合同协议书、离职申请书


1. 劳动合同解除通知书，原则上可以签， 但最好加上一句话：本人收到xx通知，但本人对内容不认可，本人签字仅代表本人收到了xx通知。

2. 解除劳动合同协议书，如果对协议内容比较满意，可以签。注意两点，一是公司要先盖章，二是我一定要拿到一份盖完章的协议

3. 离职申请书，千万不能签。不存在”离职申请“。


# 五、离职证明的内容

1. 离职证明上不能写任何对员工的负面评价，更不能贬低员工人格。如“员工不胜任工作”的说法都是不能写的。



# 六、哪些情况没有经济补偿

1. 自愿辞职
2. 劳动者有过错被开除，如违规、违纪、违法
3. 高频续订被拒，如每年续一次劳动合同，维持原工资、原岗位，但张三不同意，这种和自愿辞职的性质差不多
4. 公司设立的公益性岗位，辞退员工时不用给经济补偿
